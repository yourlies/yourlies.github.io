如何安装python环境

笔者所使用的版本，以及后续教程均采用python3.8.10版本，因此建议读者也安装此版本，以免产生不必要的版本冲突问题

  • 1. 进入python官网python3.8.10，选择Windows installer (64-bit)下载并安装
  • 2. 右键屏幕底部任务栏win图标，选择windows终端（管理员）
  • 3. 输入python --version

如果出现以下提示则说明环境已成功安装

PS C:\Users\****> python --version
Python 3.8.10

否则说明python没有被成功设置识别，需要配置环境变量，如何配置环境变量

什么是变量

变量来源于数学，是计算机语言中能储存计算结果或能表示值的抽象概念

简单的说就是，定义一个用来储存值的记号

我们在编辑器中输入

# #称之为注释，没有实际意义，只是单纯用于解释说明
# 定义一个名为title的变量，并对其赋值hello world
title = "hello world"
# 打印这个变量，输出结果是hello world
print(title)
# 定义一个名为number的变量，并对其赋值为1
number = 1
# 重新赋值number的变量，新的值为number + 1
number = number + 1
# 打印这个变量，1+1=2，最后输出结果应该是2
print(number)

什么是函数

简单来说，就是y=f(x)，我们定义一个式子f(x)，只要我们把x代入，就会得到y

同理我们新建一个文件function.py

并在编辑器中输入

# 在python中，定义函数的方式如下，照抄即可
# 其中counter为函数名，number称之为参数名，都是可以自己随便取的
# 可以把这个函数简单理解成f(x) = x + 1
def counter(number):
    print(number + 1)


# 调用这个函数，并传入参数10，运行结果为输出11
counter(10)

切换至终端并输入，会得到如下结果

PS C:\Users\95814\Documents\lies\python-script> python .\function.py
11

实际演示效果
