什么是数组

我们先来回忆一下高中数学集合的知识

关于集合的最简单的说法就是，集合是“确定的一堆东西”，集合里的“东西”则称为元素，是由一个或多个确定的元素所构成的整体

数组是一个用于存放元素的容器，和数学上的集合差不多。现在想象有一辆高铁，现在有10个乘客乘车，我们通过车票座位来对每个乘客进行区分

# 通过以下的方式定义一个数组
HighSpeedTrainSeats = ['嬴政', '赵高', '李斯', '刘邦','曹操', '刘备', '孙权', '诸葛亮', '张飞', '关羽']

由于计算机数组从0开始算起，因此第0号座位的乘客，HighSpeedTrainSeats[0]是“嬴政”

什么是变量

变量来源于数学，是计算机语言中能储存计算结果或能表示值的抽象概念

简单的说就是，定义一个用来储存值的记号

# 定义一个名为title的变量，并对其赋值hello world
title = "hello world"
# 打印这个变量，输出结果是hello world
print(title)
# 定义一个名为number的变量，并对其赋值为1
number = 1
# 重新赋值number的变量，新的值为number + 1

# 打印这个变量，1+1=2，最后输出结果应该是2
print(number)

什么是函数

简单来说，就是y=f(x)，我们定义一个式子f(x)，只要我们把x代入，就会得到y

同理我们新建一个文件function.py

并在编辑器中输入

# 在python中，定义函数的方式如下，照抄即可
# 其中counter为函数名，number称之为参数名，都是可以自己随便取的
# 可以把这个函数简单理解成f(x) = x + 1
def counter(number):
    print(number + 1)


# 调用这个函数，并传入参数10，运行结果为输出11
counter(10)

切换至终端并输入，会得到如下结果

PS C:\Users\95814\Documents\lies\python-script> python .\function.py
11

实际演示效果
